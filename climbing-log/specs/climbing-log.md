# Project: Climbing Log

## What

A simple app that allows me to track which routes I climbed.

## Main parts

- Form to input and edit an ascent (name of the route, area, grade, climbing style, date)
- Overview over all my ascents

## Style rules

- main colors: dark, blue
- modern design, with straight lines, minimal graphical elements
- easy and obvious to use

## Success criteria

- Entered ascents are saved and can be viewed in a overview page
- The overview page allows to search for recorded ascents
